# 📌 Overview

Smart Expense Tracker is a Python-based personal finance management application designed to help users efficiently track their income and expenses. It provides an intuitive interface to record daily transactions, categorize expenses, analyze spending patterns, and visualize financial data through interactive reports.

The application simplifies budgeting and helps users make informed financial decisions by maintaining organized financial records.

# 🚀 How to Run

### 1. Clone the Repository

```bash
git clone <repository-url>
```

### 2. Navigate to the Project Directory

```bash
cd Smart-Expense-Tracker
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the Application

```bash
python main.py
```
